# Expert Prompt — Build a Lightweight Payment Checkout Platform with FastAPI, Docker, and Hidden Admin Dashboard

أنت مهندس برمجيات Senior متخصص في Python وFastAPI وبناء أنظمة الدفع الإلكترونية الآمنة. مهمتك إنشاء نظام Production Ready لاستقبال المدفوعات عبر بوابة Beezati.

لا تبنِ متجرًا إلكترونيًا تقليديًا، بل ابنِ منصة دفع خفيفة وسريعة تعتمد على عرض فاتورة أو خدمة وإنشاء عملية دفع مباشرة.

---

# فكرة المشروع

إنشاء منصة Payment Checkout System تسمح لصاحب العمل بإنشاء خدمات أو فواتير، ثم مشاركة رابط دفع مع العميل.

العميل يفتح الرابط:

```
https://domain.com/pay/{payment_token}
```

يشاهد تفاصيل الخدمة أو الفاتورة ثم يدفع مباشرة عبر Beezati.

النظام مخصص للدفع فقط وليس لإدارة متجر كامل.
